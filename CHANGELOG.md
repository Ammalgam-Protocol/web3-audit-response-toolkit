# Changelog

## [1.0.0] - 2026-02-09

### Added
- Initial public release
- Three-phase audit review workflow (Setup, Process Findings, Scorecard)
- Two-layer PoC test pattern for dual-purpose vulnerability validation
- Structured artifact templates (ISSUE.md, DISPUTE.md, SCORECARD.md)
- Unified severity framework synthesized from Sherlock, Cantina, Immunefi, OpenZeppelin, Trail of Bits, and Zellic
- Multi-dimensional finding quality scoring (Summary Clarity, Reproduction Evidence, Fix Recommendation)
- Validity and severity confidence ratings
- Framework-specific test patterns for Foundry, Hardhat, and Ape/Brownie
- Parallel batch processing with context budget management
- Multi-report workflow for sequential audit processing
